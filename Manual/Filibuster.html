Filibuster Settings

This mod requires a microphone. It will look for the default recording device and listen to it to determine the volume.

There is a modSettings.json file in the mod's folder (SteamLibrary\steamapps\workshop\content\341800\739663396) to customize the thresholds. You can open this file in a text editor to edit the settings. Restart the game if you are changing these values.

This is the default for the file:
{"MicThreshold": 25.0, "FailureThreshold": 3}

MicThreshold - a value from 0.0 - 100.0 to adjust for your microphone.
FailureThreshold - an integer value for the number of seconds of failing the mic check before a strike. I recommend this stay in the range of 1 - 10.
