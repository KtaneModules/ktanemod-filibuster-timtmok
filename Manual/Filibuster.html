Filibuster Settings

This mod requires a microphone. It will look for the default recording device and listen to it to determine the volume.

There is a settings.json file in the mod's folder to customize the failure volume threshold. You can open this file in a text editor and set a value from 0.0 - 100.0 to adjust for your microphone. The setting is read when the mission starts.
